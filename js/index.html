hypnospiral visualizer source code

Custom JS:

  • main.js - The main entry point.
  • state.js - The HSState system, used to keep track of spiral settings.
  • state_color.js - Links state to the color picker.
  • state_dom.js - Links state to sliders and page text.
  • state_in_url.js - Saves and loads state from the page's URL.
  • subliminal.js - The subliminal messaging system.
  • control_toggle.js - Allows toggling the control panel.
  • fullscreen.js - Handle entering and exiting fullscreen mode.
  • gl_shaders.js - Helpers for GLSL shader compilation.
  • gl_geom.js - OpenGL geometry generation.
  • gl_canvas.js - Handles the OpenGL canvas
  • state_gl.js - Links GL uniforms to state.
  • dyslexia.js - Handles toggling of dyslexia font.

GLSL Shaders:

  • spiral.frag - The fragment shader, where the magic happens.
  • spiral.vert - A simple vertex shader.

Third-party JS:

  • iro.js - The iro.js color picker library.
